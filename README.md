# Habit Tracker

PWA для отслеживания привычек с синхронизацией между устройствами через [gyxi](https://gyxi.com).

🌐 **Live:** https://xelay.github.io/habbit-tracker

## Стек

- React + Vite + TypeScript
- Tailwind CSS v3 + shadcn/ui
- wouter (hash routing)
- PWA (Service Worker + Web App Manifest)
- Синхронизация: gyxi DB

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
# Собранный фронтенд — в dist/public/
```

## Ветки

- `main` — собранный фронтенд (GitHub Pages)
- `source` — исходный код
